# Curso de Git e GitHub

## Programa Santander Coders 2024 - Ada Tech
## Aprendendo Git 

### Comandos:

**Ajuda nos comandos**

`git help {comando}`\
`git {comando} --help`\
`man git- {comando}`\


**Definir o seu nome de usuário e endereço de e-mail**

`git config`

Ex:
~~~
git config --global user.name "John Doe"
git config --global user.email johndoe@example.com
~~~
